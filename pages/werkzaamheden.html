Werkzaamheden

In mijn stageperiode heb ik aan meerdere mooie projecten mogen meewerken. Ik heb vanaf de basisstyling van het Fusion framework (gebaseerd op Bootstrap) een groot deel van de styling gemaakt van Revalidatie Nederland, Achterstandsfondsen en Ondernemend Hoeksche Waard.

* Omdat deze websites nog niet online staan, kan ik het beeldmateriaal hiervan nog niet delen op deze site, maar heb ik wel als bijlage gedeeld op DLO.

Naast deze projecten heb ik ook mogen leren om te gaan met CSS frameworks. Ik heb een begin gemaakt aan het omzetten van het Bootstrap framework naar het snellere, makkelijkere, specifiekere Tailwind.

Tot slot heb ik de momenten dat er amper werk op tafel lag op mogen vullen met udemy cursussen. Ik heb gewerkt aan JavaScript en aan CSS animaties.

  • Revalidatie Nederland

    Revalidatie Nederland is een doelgerichte en proactieve branchevereniging. De vereniging behartigt de belangen van haar leden, die medisch specialistische revalidatiezorg bieden.

    Revalidatie Nederland is het eerste project waar ik aan heb meegewerkt binnen mijn stage bij Yard. Ik moest nog enorm wennen aan het schrijven van code op hoog niveau, omdat ik bij CMD nog lang niet zo ver was. Ik heb voor deze website verschillende componenten gestyled:

    • Navigatiebalk
    • Picture Quote

    • Berichtenlijsten
    • Filters

  • Tailwind

    Gedurende de stage heb ik leren werken met Bootstrap, een gratis open-source CSS framework dat in 2011 is bedacht. Een CSS framework houdt in dat je in de HTML pre-existing classes toe kunt voegen voor van alles: kleuren, padding, flex, grid, borders, etc. Met dit framework kun je best veel, maar je moet als developer nog een hoop extra CSS bijschrijven.

    ‘Fusion-whitelabel’, wat het standaard template is dat Yard gebruikt voor nieuwe sites, maakt momenteel gebruik van Bootstrap. Dit wil Yvette om gaan zetten naar Tailwind. Tailwind is een nieuwer, veel specifieker framework. Hiermee kun je hele kleine, specifieke classjes op al je HTML elementen gaan zetten, waardoor je eigenlijk bijna geen CSS meer zelf hoeft te schrijven.

    Ik ben begonnen met het lezen van de documentatie, hier is ontzettend veel over te vinden - bijna te veel.

    De docs van Tailwind hebben een goede zoekfunctie. Doordat ik de voorgaande weken al met Bootstrap heb gewerkt, is de logica in Tailwind een stuk makkelijker te vinden.

    De manier van opschrijven was nog wat lastig. De overstap van SASS naar CSS was nog een twijfelfactor, en Tailwind biedt een hele korte maar redelijk onoverzichtelijke manier van opschrijven met @apply. Dit zorgt ervoor dat je je CSS niet uit hoeft te schrijven, maar je de relevante Tailwind classes kunt toevoegen aan de elementen door @apply vooraan de regel te zetten.

  • Animatie

    Toen het hele kantoor naar Berlijn vertrok, mocht ik mezelf een week vermaken met wat zelfstudie. Toen besloot ik de 'Advanced CSS Animations' cursus te doen, omdat ik animaties heel interessant vind en ik denk dat dit goed aansluit bij CMD als opleiding.
